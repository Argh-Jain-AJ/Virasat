Virasat – Digital Family Legacy Platform
Virasat is a full-stack web application designed to preserve, visualize, and manage generational family history through an interactive and modern digital experience. The platform allows users to create dynamic family trees, manage relationships between members, document memories and timelines, and collaborate securely with relatives in real time. Built with scalable architecture and production-ready deployment support, the project combines graph visualization, secure authentication, and structured relational data management into one unified platform.

Features
Interactive family tree visualization using React Flow and Dagre
Dynamic node-based relationship management
Detailed member profiles with memories and timelines
Secure authentication with JWT and bcrypt
Role-based collaboration and family access management
AI-assisted biography generation for family members
Export family trees as high-quality PNG images
Dockerized full-stack deployment setup
PostgreSQL-backed normalized relational database structure
